# FilterWheels

Electronics and harware used to automate the process of moving optical filter wheels during experiments.

See documentation.pdf for full information.

## yaq support

The daemons in this repository (`./wright-filter-wheels`) in outdated and unmaintained; instead, use the wright-stepper-motor-box kind found in [`yaqd-wright`](https://github.com/wright-group/yaqd-wright).
